# FarmBridge
Farm Bridge is a hyperlocal farmer-to-consumer e-commerce platform designed to help farmers sell fresh produce directly to nearby consumers. The platform removes middlemen, ensures fair pricing for farmers, and provides consumers with fresh, locally sourced products.

## Features
- Farmer and consumer registration
- Product listing with price and quantity
- Pre-order based purchasing
- Location-based farmer discovery
- Order status and delivery tracking
- Simple admin control for management

## Tech Stack
- Frontend: React, Tailwind CSS  
- Backend: Node.js, Express  
- Database: MongoDB  
- Maps & Tracking: Google Maps API  

## System Overview
The platform follows a client-server architecture where farmers and consumers interact through a web interface. The backend manages authentication, product data, orders, and location-based matching, while the database stores user and order information.
